How to Play

Welcome to the Lucky Number Game! Here's how to play:

  • Choose a number between 0-9 from the number grid
  • Select your bet amount (10-100 coins)
  • If the random number matches your choice, you win!
  • Payout multipliers vary based on the number selected

Game Rules

  • Minimum bet: 10 coins
  • Maximum bet: 100 coins
  • Numbers have different multipliers (shown below each number)
  • You start with 100 coins
  • If you run out of coins, you need to login/register

Payout Multipliers

Each number has different payout rates:

  • 0: x10
  • 1: x6
  • 2: x5.5
  • 3: x5
  • 4: x4.5
  • 5: x4
  • 6: x3.5
  • 7: x3
  • 8: x2.5
  • 9: x2
